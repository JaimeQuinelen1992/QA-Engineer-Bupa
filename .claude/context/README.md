# context/

Contexto activo del proyecto que Claude debe leer al iniciar una sesion.

## Uso
- `current-sprint.md` — objetivos y alcance del sprint en curso
- `environment.md` — URLs, puertos, y datos de ambientes (sin credenciales)
- `stack.md` — versiones exactas de dependencias clave

Claude lee estos archivos para entender el estado actual sin necesidad de preguntar.
