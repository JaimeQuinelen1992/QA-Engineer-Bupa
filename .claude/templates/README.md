# templates/

Plantillas reutilizables para generar artefactos QA consistentes.

## Plantillas disponibles
- `test-case.md` — estructura estandar de caso de prueba
- `bug-report.md` — formato de reporte de defecto
- `sdd.md` — estructura de documento SDD
- `cypress-spec.cy.ts` — spec base de Cypress con hooks y estructura
