# templates/

Plantillas reutilizables para generar artefactos QA consistentes.

## Plantillas disponibles
- `test-plan-manual.md` — plan de pruebas manuales completo (objetivo, alcance, DoR/DoD, estrategia por categoria, TCs con detalle, registro de ejecucion y resumen)
- `test-plan-automatizado.md` — plan de pruebas automatizadas para flujos criticos con Cypress (criterio de seleccion, spec structure, configuracion de ambiente, integracion pipeline, quality gate)
- `bug-report.md` — formato de reporte de defecto
- `sdd.md` — estructura de documento SDD
- `cypress-spec.cy.ts` — spec base de Cypress con hooks y estructura
