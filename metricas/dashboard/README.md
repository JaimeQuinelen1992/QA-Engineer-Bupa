## Columnas del Dashboard

| Columna | Descripción |
|---------|-------------|
| Semana | Código ISO (ej: 2026-W22) |
| Fecha | Viernes de esa semana |
| Test Coverage | % con semáforo (rojo < 90%) |
| Pass/Fail Rate | % con semáforo (rojo < 95%) |
| Automation Coverage | % con semáforo (rojo < 70%) |
| Escaped Defects | Número absoluto (rojo si > 0) |
| Defect Resolution | % con semáforo (rojo < 95%) |
| Notas | Observaciones del QA Lead |
